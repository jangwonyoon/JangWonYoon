### 👋 About Me

안녕하세요. 프론트엔드 경력 5년차 개발자 윤장원입니다.

지난 4년간 KINX에서 클라우드 콘솔 UI/UX를 처음부터 구축하고 운영하는 경험을 쌓았습니다.
React, Next.js, TypeScript, TailwindCSS를 기반으로 복잡한 클라우드 리소스를 시각화하고,
대규모 B2B 환경에 맞는 도메인 기반 아키텍처, 상태관리 전략, 디자인 시스템, 성능 최적화 구조를 직접 설계·도입했습니다.
이 과정에서 유지보수성과 확장성을 고려한 사고방식이 자연스럽게 몸에 배었고,
문제를 정의하고 개선하는 흐름을 체계적으로 다루는 역량을 키울 수 있었습니다.

현재는 카카오헬스케어에서 의료 데이터 플랫폼(HRS)의 프론트엔드 신뢰성·관측 계층을 구축하고 있습니다.
대규모 의료 데이터를 다루는 환경에서 성능 모니터링, 오류 추적, 사용자 여정 기반 트레이싱, 멀티테넌트 구조 최적화 등을 포함하여
플랫폼 전반의 품질을 높이는 프론트엔드 Observability 시스템을 설계하고 있습니다.

아키텍처 설계, 성능 및 관측성 개선, 사용자 경험 향상에 깊은 관심이 있으며
배운 내용을 정리하고 공유하는 활동을 꾸준히 이어가고 있습니다.
